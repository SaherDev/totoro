# Implementation Plan: Wire Recall Proxy to totoro-ai

**Branch**: `009-recall-proxy` | **Date**: 2026-03-31 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/009-recall-proxy/spec.md`

## Summary

Replace the `RecallService` 501 stub with a working proxy to totoro-ai's `POST /v1/recall`. Wires `IAiServiceClient` into the recall domain, adds `recall()` to the client interface and implementation, updates DTOs to match the API contract, and produces Bruno + Jest test coverage.

No database changes. No streaming. Pure synchronous JSON proxy — same pattern as `extractPlace()`.

## Technical Context

**Language/Version**: TypeScript 5.x / Node 20 LTS
**Primary Dependencies**: NestJS 11, `@nestjs/axios` (HttpService), `class-validator`, `class-transformer`
**Storage**: None — recall is a pure proxy (no DB writes)
**Testing**: Jest (unit), Bruno (manual API)
**Target Platform**: Railway (NestJS service)
**Project Type**: NestJS web service — thin gateway
**Performance Goals**: Recall response ≤ 20s (per api-contract.md timeout policy)
**Constraints**: AxiosError propagates to `AllExceptionsFilter`; no custom error handler in service
**Scale/Scope**: Single endpoint change; touches 7 existing files + 2 new files

### Phase 1: Interface + Client (ai-service layer)

**1.1 — ai-service-client.interface.ts**

```typescript
export interface AiRecallPayload {
  user_id: string;

export interface AiRecallPlace {
  place_id: string;
  place_name: string;
  address: string;
  cuisine?: string | null;
  price_range?: string | null;
  source_url?: string | null;
  match_reason: string;
  saved_at?: string | null;
}

export interface AiRecallResponse {
  results: AiRecallPlace[];
  total: number;
}
```

Add to `IAiServiceClient` interface:

```typescript
recall(payload: AiRecallPayload): Promise<AiRecallResponse>;
```

**1.2 — ai-service.client.ts**

Add `recall()` method (same pattern as `extractPlace()`):

```typescript
async recall(payload: AiRecallPayload): Promise<AiRecallResponse> {
  const response = await firstValueFrom(
    this.httpService.post<AiRecallResponse>(
      `${this.baseUrl}/v1/recall`,
      payload,
      { timeout: 20000 }
    )
  );
  return response.data;
}
```

Import `AiRecallPayload`, `AiRecallResponse` from interface file.

**1.3 — ai-service.client.spec.ts**

```typescript
})

```

**3.2 — recall.service.ts**

Replace stub with:

```typescript
@Injectable()
export class RecallService {
  constructor(
    @Inject(AI_SERVICE_CLIENT) private aiClient: IAiServiceClient
  ) {}

  async recall(dto: RecallRequestDto): Promise<RecallResponseDto> {
    const payload: AiRecallPayload = {
      user_id: dto.user_id,
      query: dto.query,
    };
    return this.aiClient.recall(payload);
  }
}
```

**3.3 — recall.controller.ts**

- Remove `@HttpCode(501)` decorator
- Remove `NotImplementedException` import
- Remove `@Request()` and `req.user` usage
- Pass full `dto` to service

```typescript
@Controller('recall')
export class RecallController {
  constructor(private readonly recallService: RecallService) {}

  @Post()
  async recall(@Body() dto: RecallRequestDto): Promise<RecallResponseDto> {
    return this.recallService.recall(dto);
  }
}
```
